## Archive Policies

### When to Archive
- Notebooks replaced by newer implementations
- Experimental notebooks after feature integration
- Development artifacts after milestone completion
- Redundant notebooks after consolidation

### Archive Process
1. Create timestamped directory
2. Copy notebook with outputs preserved
3. Include associated data files
4. Generate comprehensive README
5. Update this index
6. Remove from active development path

### Recovery Process
Each archived notebook includes recovery instructions in its README file.
